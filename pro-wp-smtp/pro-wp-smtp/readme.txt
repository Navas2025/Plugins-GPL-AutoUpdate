Plugin SMTP completo para WordPress con todas las funcionalidades profesionales activadas.

== Description ==

Pro WP SMTP es un plugin completo para configurar el envío de correos electrónicos desde WordPress usando SMTP. Incluye soporte para servidores SMTP corporativos y servicios populares.

= Características Principales =

* Configuración SMTP corporativa completa
* Host, puerto, usuario, contraseña personalizables
* Encriptación SSL/TLS
* Soporte para múltiples proveedores (Gmail, SendGrid, Mailgun)
* Email logging y tracking
* Reportes y estadísticas
* Test de envío de correos
* Todas las funciones PRO activadas

== Installation ==

1. Subir el plugin a través del menú 'Plugins > Añadir nuevo' de WordPress
2. Activar el plugin
3. Ir a 'Pro WP SMTP > Configuración' para configurar

== Frequently Asked Questions ==

= ¿Cómo configuro mi correo corporativo? =

1. Ir a Pro WP SMTP > Configuración
2. Seleccionar "Otro SMTP"
3. Ingresar los datos de tu servidor SMTP (host, puerto, usuario, contraseña)
4. Guardar y probar el envío

= ¿Qué puerto SMTP debo usar? =

- Puerto 587: Para TLS (recomendado)
- Puerto 465: Para SSL
- Puerto 25: Sin encriptación (no recomendado)

= ¿Funciona con Gmail? =

Sí, el plugin incluye soporte específico para Gmail con OAuth2.
